Belvedere
=========

Belvedere sets up rules for taking  actions on files (move, copy, delete, etc) based on the name of a file, its extension, size, age, and more.

## System Requirements

- Windows 7, 8, 10, or 11
- AutoHotkey v1.1 or later (required for running the .ahk script directly)

## Installation

For Windows 11 users, please ensure you run the installer with administrator privileges to ensure proper integration with the system.

For usage details, see [Lifehacker](http://lifehacker.com/341950/belvedere-automates-your-self+cleaning-pc)
